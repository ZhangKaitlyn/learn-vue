单文件组件

介绍

使用Vue.component来定义全局组件缺点：

  • 全局定义 强制要求每个component中的命名不得重复
  • 字符串模板 缺乏语法高亮，在HTML有多行的时候，需要用到丑陋的 \
  • 不支持CSS 意味着当HTML和JavaScript组件化时，CSS明显被遗漏
  • 没有构建步骤 限制只能使用HTML和ES5 JavaScript，而不能使用预处理器
